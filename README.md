# Mahatma Gandhi Museum Website – README

## Project Overview

The **Mahatma Gandhi Museum Website** is an informational website created to present the history, heritage, and collections of the Mahatma Gandhi Museum located in Mangaluru, Karnataka. The website provides visitors with an engaging digital experience through images, galleries, historical information, and interactive visual elements. Its aim is to promote awareness of Gandhian values, cultural heritage, and museum collections.

## Features

• **Hero Section with Video** – Introductory section with a video preview about the museum.
• **History & Heritage Section** – Explains the historical significance of the museum and Mahatma Gandhi’s connection with the institution.
• **Interactive Image Galleries** – Displays museum collections such as sculptures, artifacts, technology, and natural specimens.
• **Looping Image Collections** – Some gallery items automatically rotate through multiple images.
• **Cabinet of Curiosity Slider** – Image slider showcasing unique museum artifacts.
• **Parallax Background Section** – Creates a visual depth effect for the Cabinet of Curiosity section.
• **Plan Your Visit Section** – Includes location map, visiting hours, and visitor information.
• **Educational Value Section** – Highlights the museum’s role in value-based education.
• **Navigation Info Box** – Hover navigation reveals quick information about sections.
• **Responsive Design** – Optimized for mobile, tablet, and desktop devices.

## Technologies Used

HTML5 – Website structure
CSS3 – Styling, layout, responsive design, and parallax effects
JavaScript – Interactive features such as sliders, video control, navigation info box, and image loops
Google Fonts – Typography styling (Cormorant Garamond)

## How to Run the Project

1. Download or clone the project files.
2. Ensure all files and the **src image/video folder** are in the same project directory.
3. Open the project folder in a code editor (e.g., VS Code).
4. Open **index.html** in any modern web browser.

## Project Structure

index.html – Main webpage containing all sections of the museum site
museum.css – Stylesheet controlling layout, design, and responsiveness
src/ – Folder containing all images and video used in the website
